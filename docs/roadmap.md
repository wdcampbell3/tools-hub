# Project Roadmap

This document tracks the high-level goals and status of the SaaS Starter project.

## Completed Milestones
- [x] **Project Inception**: Initial SvelteKit + Tailwind setup.
- [x] **Core Features**: Authentication, Dashboard, Billing, Blog, Marketing Landing Page.
- [x] **Backend Migration**: Successfully migrated from Supabase to **Firebase** for Auth and Database (Firestore).
- [x] **Deployment Update**: Switched hosting strategy from Cloudflare to **Netlify**.

## Current Focus
- [ ] **Documentation**: Establishing comprehensive docs for AI-assisted development.
- [ ] **Testing**: Ensuring full coverage for new Firebase backend.

## Future Goals
- [ ] **Admin Analytics**: Add dashboard charts for user growth and MRR.
- [ ] **Mobile App**: Explore Capacitor or React Native wrapper.
- [ ] **Internationalization**: Add i18n support.
- [ ] **AI Features**: Integrate basic AI capabilities (e.g., text generation assistant).
